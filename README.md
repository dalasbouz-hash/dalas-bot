# 🤖 دليل تشغيل وتدبير بوت الحماية (Discord Moderation Bot)

هاد البوت كيمسح **روابط السيرفرات الأخرى** و**المحتوى غير اللائق (+18)** تلقائياً وكيعطي إنذار (Warning) للمخالفين.

---

## 📌 الخطوة 1: إعادة تعيين الـ Token (ضروري جداً ⚠️)
حيث حطيتي الـ Token فالتشات، راه ولا مكشوف وأي واحد يقدر يتحكم بالبوت ديالك. خاصك تغيرو دابا:
1. دخل لـ [Discord Developer Portal](https://discord.com/developers/applications).
2. كليكي على البوت ديالك.
3. دخل لـ **Bot** من القائمة اللي على اليسار.
4. كليكي على زر **Reset Token** وانسخ الـ Token الجديد.
5. افتح ملف `.env` في المجلد `c:\BOT` وحط التوكن الجديد بلاصة `YOUR_DISCORD_TOKEN_HERE`.

---

## 📌 الخطوة 2: تفعيل الـ Intents فـ Discord Developer Portal
باش البوت يقدر يقرا الرسائل ويمسحها، خاص تفعل ليه الصلاحيات:
1. فـ **Developer Portal** (صفحة البوت ديالك).
2. هبط لتحت فصفحة **Bot** حتى تلقى **Privileged Gateway Intents**.
3. فعل هاد الخيارات بثلاثة:
   - **Presence Intent**
   - **Server Members Intent**
   - **Message Content Intent** (مهم جداً!)
4. كليكي على **Save Changes**.

---

## 📌 الخطوة 3: دخل البوت للسيرفر ديالك (Invite)
1. فـ **Developer Portal**، دخل لـ **OAuth2** ثم **URL Generator** (على اليسار).
2. فخانة **Scopes**، علم على: `bot`.
3. فخانة **Bot Permissions** (الصلاحيات)، علم على:
   - `Manage Messages` (مسح الرسائل)
   - `Send Messages` (إرسال الرسائل)
   - `Read Message History` (قراءة تاريخ الرسائل)
4. غادي يعطيك واحد الرابط لتحت (Generated URL)، نسخه وافتحه فالمتصفح باش تضيف البوت للسيرفر ديالك.

---

## 📌 الخطوة 4: تفعيل ميزة منع الصور الإباحية (+18) فالسيرفر فابور
ديسكورد فيه فلتر قوي ومجاني للصور الإباحية، دير هاد الخطوات فالسيرفر ديالك:
1. افتح **Server Settings** (إعدادات السيرفر).
2. دخل لـ **AutoMod**.
3. فعل **Block Explicit Media** (منع المرفقات الصريحة/الإباحية).
4. هاد الميزة كتكشف الصور الإباحية مباشرة وكتمسحها قبل ما كاع توصل للشات!
5. البوت ديالنا كيكمل الخدمة بمسح الكلمات الإباحية والإنذار بالدارجة.

---

## 📌 الخطوة 5: تشغيل البوت
1. افتح Terminal (PowerShell أو CMD) فالمجلد `c:\BOT`.
2. شغل الأمر التالي لتشغيل البوت:
   ```bash
   node index.js
   ```
3. غادي تطلع ليك رسالة: `✅ Bot is online!` يعني البوت خدام وكيراقب السيرفر دابا.

---

## ⚙️ الإعدادات فملف `.env`
* `DISCORD_TOKEN`: التوكن ديال البوت.
* `ALLOWED_INVITE_CODES`: إلا بغيتي شي رابط ديال السيرفر ديالك ما يتمسحش، حط الكود ديالو هنا (مثال: `ALLOWED_INVITE_CODES=aBcd123,eFgh456`).
